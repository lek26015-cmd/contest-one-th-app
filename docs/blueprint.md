# **App Name**: ContestOne

## Core Features:

- Competition Submission: Allow users to submit new competition details including title, description, deadline, and category.
- Competition Browsing: Display all available competitions in a list format.
- Competition Filtering: Enable users to filter competitions by category and deadline.
- Detailed View: Show detailed information for each competition, including all submitted details.

## Style Guidelines:

- Primary color: Vibrant teal (#008080) to create a modern and engaging look, per user request.
- Background color: Light gray (#E0E0E0) to provide a neutral backdrop that highlights the teal accents, per user request.
- Accent color: A slightly darker teal (#006969) to be used for interactive elements.
- Body and headline font: 'Inter' for a modern, machined, objective, neutral feel.
- Use clean and simple icons to represent competition categories.
- Use a grid-based layout for displaying competitions to ensure a clean and organized look.
- Subtle transitions and animations for filtering and displaying competition details.
